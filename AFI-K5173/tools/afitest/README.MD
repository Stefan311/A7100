# AFITest
Mit AFITest lassen sich die Hardware-Funktionen des IDE-Kontrollers interaktiv testen.

# Benutzung
ist ähnlich zur A7150-Version, und dort beschrieben.

# Kompilierung
1. Turbo-Pascal 3, AFITEST.PAS, KESPROG.INC, und KES.PAS auf eine Diskette kopieren.
2. Auf dem A7100 `turbo` ausführen
3. `y` (Fehlermelde-Datei laden)
4. `m` und `afitest.pas` (Dateiname Hauptdatei angeben)
5. `o` dann `c` dann `q` (Kompiler-Ziel auf CMD-Datei stellen)
6. `c` Kompilierung durchführen.
